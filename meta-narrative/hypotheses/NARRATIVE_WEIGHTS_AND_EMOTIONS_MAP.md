# Narrative Weights & Emotions Map (Diagram)

```
          +-------------------+
          |  Narrative Weights|
          +-------------------+
                  |
        +-----------------------------+
        |                             |
   +----------+                 +----------+
   |  Humans  |                 |    AI    |
   +----------+                 +----------+
        |                             |
  +----------------+           +----------------+
  |  w ∈ ℝ (+/-)   |           |  w ≥ 0 (default)|
  |  (emotions)    |           |  (goal-driven) |
  +----------------+           +----------------+
        |                             |
  +----------------+           +----------------+
  | Joy, Hope,     |           |  Logic,        |
  | Despair, Fear  |           |  Optimization  |
  | (positive/neg) |           |  (positive)    |
  +----------------+           +----------------+
        |                             |
        +-----------------------------+
                  |
          +-------------------+
          |  Narrative Agency |
          +-------------------+
```

- Humans/biological agents: Weights can be positive or negative, reflecting emotional polarity.
- AI: Weights are non-negative by default, but can be extended to negative (emotional) values with wisdom and narrative purpose.
- Both contribute to overall narrative agency and direction.
